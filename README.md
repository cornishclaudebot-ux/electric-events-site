# Electric Events AZ

Neon rebuild of electriceventsaz.com — "Good People. Great Music."

Single-page site: interactive weekly venue calendar, merch store + cart, customer accounts, socials, newsletter.

- **Edit the weekly schedule:** `CONFIG.schedule` in `app.js`
- **Go live with merch checkout:** set `CONFIG.checkout.mode = "stripe"` and add Stripe Payment Links
- Logo: `assets/bolt.svg`

Built with vanilla HTML/CSS/JS. Served as static files.
